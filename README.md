# CyberPhish

CyberPhish is an AIO Web/Social Phishing tool to grab credentials from targetted victims.

## Getting Started

To get started, follow the instructions below:

```
Clone this repo to your local machine - "git clone https://github.com/derickrn/cyberphish"
Browse to folder it was saved to 
Run CyberPhish with the following command - "bash cyberphish.sh"
```
